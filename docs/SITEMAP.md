## 9. Notes | ملاحظات

* Arabic pages should use RTL layout.
* English pages should use LTR layout.
* Each dynamic page should have a unique slug.
* Old website links should be redirected to the new URLs.
* All important pages should include SEO metadata.
* Sensitive forms should not expose submitted data publicly.
